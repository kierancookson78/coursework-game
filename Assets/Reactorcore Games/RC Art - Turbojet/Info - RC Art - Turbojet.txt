Reactorcore Art (RC Art)
==
PACK: Turbojet

--

Pixel-art sprites pack depicting a fighter turbojet plane - including 6 color variants, each having a landed-, VTOL- and flight-mode sprites.

--

Trivia: Inspired by SWAT Cats and A-10 Warthog.

--

By default in image import settings, all sprites/textures have been set to 32 pixels per unit and using a point filter so they appear crisp.
